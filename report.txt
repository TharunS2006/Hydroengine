SECTION 1: PROJECT IDEA
================================================================

The user (TharunS2006) proposed building an engine that can run
on BOTH petrol and a hydrogen-oxygen tank, with the ability to
switch between the two fuels on command.

================================================================
SECTION 2: ENGINE TYPES EXPLORED
================================================================

Four main engine types were identified as suitable:

1. Modified Internal Combustion Engine (ICE)
   - Retrofitted petrol engine to also burn hydrogen
   - Example: BMW Hydrogen 7

2. Dual-Fuel Engine
   - Injects hydrogen alongside petrol simultaneously
   - Most practical and flexible option

3. Rotary (Wankel) Engine
   - Mazda tested hydrogen + petrol dual-fuel on rotary engines
   - Flexible design for multiple fuels

4. Multi-Fuel Engine (From Scratch)
   - Designed to run on petrol, hydrogen, or both
   - Example: Toyota upcoming multi-fuel engine

================================================================

================================================================

After considering budget and practicality constraints, the best

petrol engine rather than building from scratch or buying

Recommended Engine Candidates:

================================================================
SECTION 4: SWITCHING SYSTEM DESIGN
================================================================

The system allows the driver to switch between petrol and
hydrogen on command using a dashboard toggle switch.

System Flow:
- [ Petrol Tank ]  --> [ Petrol Injectors ]
                                             --> [ Engine ]
- [ H2+O2 Tank ]   --> [ H2 Injectors   ]

- A dashboard switch sends a signal to the ECU
- ECU activates/deactivates solenoid valves
- Fuel mode indicator light shows current fuel (Petrol/Hydrogen)

================================================================
SECTION 5: REQUIRED MODIFICATIONS
================================================================

Step 1 - ECU Tuning:
- Use aftermarket ECU (MegaSquirt or Arduino-based)
- Program fuel switching logic
- Adjust ignition timing per fuel type

Step 2 - Dual Injector Setup:
- Keep original petrol injectors
- Add secondary hydrogen injectors (CNG-type)
- Use solenoid valves to control fuel lines

Step 3 - Hydrogen-Oxygen Tank Setup:
- High-pressure hydrogen cylinder
- Pressure regulator for H2 flow control
- Use ambient air for oxygen (RECOMMENDED over pure O2)

Step 4 - Driver Switch:
- Dashboard toggle switch
- Wired to ECU -> solenoid valves
- Fuel mode indicator light

SECTION 7: HYDROGEN ENGINE DAMAGE RISKS & FIXES
================================================================

Risk 1 - Hydrogen Embrittlement [SEVERITY: HIGH]
- Hydrogen atoms penetrate metal making it brittle
- Fix: Use stainless steel / aluminum alloy components

Risk 2 - Higher Combustion Temperature [SEVERITY: MEDIUM-HIGH]
- Hydrogen burns hotter than petrol
- Fix: Upgrade valves, valve seats and cooling system

Risk 3 - Pre-ignition / Backfire [SEVERITY: MEDIUM-HIGH]
- Hydrogen ignites very easily and too early
- Fix: Retune ignition timing via ECU for hydrogen mode

Risk 4 - Engine Oil Degradation [SEVERITY: MEDIUM]
- Hydrogen degrades engine oil faster
- Fix: Use full synthetic oil, change oil more frequently

Risk 5 - Valve & Valve Seat Wear [SEVERITY: MEDIUM-HIGH]
- Hydrogen does not lubricate valves like petrol does
- Fix: Install hardened valve seats (same as LPG/CNG fix)

================================================================
SECTION 10: SAFETY NOTES
================================================================

- NEVER use pure H2 + O2 without proper pressure containment
- Pure H2 + O2 = rocket-level combustion, extremely dangerous
- Always test on a stationary engine bench first
- Use proper hydrogen-rated fittings and hoses
- Install a hydrogen leak detector near the engine bay
- Keep a fire extinguisher nearby during all testing
- Hydrogen is highly flammable - proper sealing is critical
